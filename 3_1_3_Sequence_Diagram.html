# Flow In 2 Words What It Means
1 Attacker → Windows OS AttackSMB Probe Attacker connects over port 445 and opens a honeyfile remotely.
2 Windows OS → Cerberus DetectEvent Fired Windows raises Security Event 5145 (shared-file access audit log).
3 Cerberus → Cerberus DetectParse Event Cerberus extracts attacker IP, username, and file name from the XML log.
4 Cerberus → Cerberus DetectScore Threat SMB_GET action gets risk score 80 → classified as CRITICAL severity.
5 Cerberus → alerts.log LogWrite Log Structured threat record (timestamp, IP, file, severity) saved to disk.
6 Cerberus → Telegram API AlertSend Alert HTTP POST to Telegram Bot API with attacker details and file name.
7 Telegram → Administrator AlertNotify Admin Admin's phone receives a real-time push notification instantly.
— Note (Admin side) Manual Step Admin decides to open the dashboard to review full event history.
8 Admin → Dashboard ReviewOpen GUI Admin launches the Streamlit web dashboard in their browser.
9 Dashboard → alerts.log ReviewRead Log Dashboard reads and parses every entry in alerts.log for display.
10 alerts.log → Dashboard ReviewReturn Data All historical intrusion records returned to the dashboard layer.
11 Dashboard → Admin ReviewShow Results Risk gauges, source-IP tables, and event timelines rendered on screen.
